# Security Policy

## Supported versions

Security fixes are applied to the latest main branch. Older releases are not maintained separately.

## Reporting a vulnerability

If you discover a vulnerability, please email **security@japantrade.dev** with the following details:

- A description of the issue and potential impact
- Steps to reproduce (commands, data samples, or screenshots where applicable)
- Your environment (OS, Python version, and any relevant configuration)

We will acknowledge receipt within five business days and will work with you to understand and validate the report. Please avoid filing public issues for security concerns until a fix is available.

## Disclosure

Once a fix is ready, we will publish details in the changelog or release notes. We may request your coordination on timing if broader ecosystem notification is needed.
